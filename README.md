# Prettify JSON

This program take JSON file and print content in terminal

# Quickstart

Run pprint_json.py

Example of script launch on Linux, Python 3.5:

```#!bash

$ python pprint_json.py
# Please enter full way to JSON file: <path to file>
[
    {
        "date": "2017-05-10",
        "title": "Conference"
    }
]

```
